Для обучения работе с базами данных использовался датасет в виде лога сервера Nginx (https://www.kaggle.com/datasets/eliasdabbas/web-server-access-logs) урезанный до первых 300 тысяч записей.

Строки данного лога распаршены и загружены в SQLite базу данных, из которой с помощью SQL запросов получены некоторые аналитические табличные и графические данные
